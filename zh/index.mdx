QitOS 是面向 agent 研究者的 torch-flavor 框架。你可以在同一个 `AgentModule + Engine` 内核上原型化方法、运行 benchmark，并用内建 `qita` 观察长时轨迹。

## 两条 authoring path

QitOS 会刻意保留两条顶层的 agent 编写路径：

- `Research-first`：手写 prompt、parser、protocol、transport 和 tool surface，直接研究与优化 kernel。
- `Preset-first`：从 family preset 和 preset tool builders 出发，快速得到稳定 baseline 与多模型切换能力。

这两条路径都应该是一等公民。

## 快速启动

<Steps>
  <Step title="安装 QitOS">
    ```bash
    pip install "qitos[models]"
    ```
  </Step>
  <Step title="设置 provider 配置">
    ```bash
    export OPENAI_API_KEY="sk-..."
    export OPENAI_BASE_URL="https://api.siliconflow.cn/v1/"
    export QITOS_MODEL="Qwen/Qwen3-8B"
    ```
  </Step>
  <Step title="运行最小 coding agent">
    ```bash
    qit demo minimal
    ```
  </Step>
  <Step title="用 qita 查看运行轨迹">
    ```bash
    qita board --logdir runs
    ```
    打开 [http://127.0.0.1:8765](http://127.0.0.1:8765)，查看 trace、replay 和导出结果。
  </Step>
</Steps>
