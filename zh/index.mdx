QitOS 提供了一套稳定的 `AgentModule + Engine` 内核，用来构建可复现的 LLM agent。无论你是在原型化 ReAct、运行 GAIA 评测，还是交付一个可长期运行的编码 agent，QitOS 都会把运行过程做成可追踪、可组合、可复现的研究对象。

## 快速启动

<Steps>
  <Step title="安装 QitOS">
    ```bash
    pip install qitos
    ```
  </Step>
  <Step title="设置 API Key">
    ```bash
    export OPENAI_API_KEY="sk-..."
    ```
  </Step>
  <Step title="运行最小示例">
    ```bash
    python examples/quickstart/minimal_agent.py
    ```
  </Step>
  <Step title="用 qita 查看运行轨迹">
    ```bash
    qita board --logdir runs
    ```
    打开 [http://127.0.0.1:8765](http://127.0.0.1:8765)，查看 trace、replay 和导出结果。
  </Step>
</Steps>
