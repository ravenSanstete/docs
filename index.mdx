## Get up and running

<Steps>
  <Step title="Install QitOS">
    ```bash
    pip install qitos
    ```
  </Step>
  <Step title="Set your API key">
    ```bash
    export OPENAI_API_KEY="sk-..."
    ```
  </Step>
  <Step title="Run the minimal example">
    ```bash
    python examples/quickstart/minimal_agent.py
    ```
  </Step>
  <Step title="Inspect your run with qita">
    ```bash
    qita board --logdir runs
    ```
    Open [http://127.0.0.1:8765](http://127.0.0.1:8765) to view trace, replay, and export your run.
  </Step>
</Steps>
